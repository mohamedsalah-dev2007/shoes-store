# 👟 مشروع متجر SoleStyle الإلكتروني - بالـ React

أهلاً بكم في مشروعي الجديد! ده عبارة عن متجر إلكتروني متكامل وحديث لبيع الأحذية بمختلف الأقسام، صممته وبرمجته بالكامل من الصفر باستخدام **React.js** عشان أطبق فيه على مفاهيم الـ State Management والـ Dynamic Filtering ومحاكاة تجربة مستخدم حقيقية وسلسة.

---

## 🔥 ميزات عملت عليها في المشروع:

* 🎯 **فلترة الأقسام فوريًا:** عملت لوجيك يخلي المستخدم يتنقل بين أقسام (الرجالي 👟، الحريمي 👗، والأطفال 🧸) في جزء من الثانية وبدون ما الصفحة تعمل ريفريش.
* 🔍 **م بحث لحظي:** ضفت شريط بحث ذكي، بمجرد ما تكتب حرف أو كلمة بيفلتر المنتجات اللي قدامك فوراً بناءً علىnpm run deploy الاسم اللي بتدور عليه.
* 🛒 **سلة مشتريات تفاعلية:** السلة بتحسب إجمالي المنتجات والكمية والأسعار تلقائيًا، ومعاها إمكانية حذف أي كوتش مش محتاجه بسهولة.
* 🧾 **فاتورة شراء مخصصة وبأرقام فريدة:** بدل رسائل الـ alert التقليدية، عملت واجهة فاتورة شيك بتظهر أول ما تضغط "تأكيد الطلب"، بتعرض رقم طلب عشوائي وتاريخ اليوم والمنتجات اللي اشتريتها.
* 📱 **باركود ذكي (QR Code) لكل فاتورة:** دمجت API خارجي بيعمل توليد لباركود ديناميكي مشفر خاص بكل فاتورة، تقدر تمسحه بكاميرا الموبايل عشان يقرا بيانات طلبك للتحقق.

---

## 🛠️ الأدوات والتقنيات اللي استخدمتها:

* **React.js** (عشان الـ Components والـ Hooks وإدارة البيانات)
* **JavaScript (ES6+)** (لوجيك الفلترة والبحث وحسابات السلة)
* **JSON** (عملت ملف منتجات كامل بالأسماء والصور والأسعار لكل قسم)
* **CSS in JS / Inline Styles** (لتنسيق واجهة متناسقة ومريحة للعين وتدعم الـ RTL)
